# dotMaxine

A Vite + React project.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
  components/
    Navbar.jsx          # Fixed top navigation
    Navbar.module.css
    HeroZoom.jsx        # Scroll-driven zoom hero section
    HeroZoom.module.css
    NextSection.jsx     # Intersection Observer reveal section
    NextSection.module.css
    Footer.jsx          # Rounded footer with big display text
    Footer.module.css
  assets/
    hero-bg.png         # Hero background image
  App.jsx
  main.jsx
  index.css             # Global reset + CSS variables
```
